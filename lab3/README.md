# Лабораторна робота 3

## Опис завдання

- Завантаження датасету з відкритого джерела (Heart Disease UCI)
- Data Cleaning: видалення пропущених значень
- Побудова 7 графіків для аналізу залежностей між атрибутами:
   - **Scatter Plot** - залежність віку від максимального пульсу
   - **Line Plot** - середній холестерин за віком
   - **Гістограма** - розподіл пацієнтів по 5 вікових діапазонах
   - **Heatmap** - кореляція між атрибутами
   - **Box Plot** - холестерин у хворих і здорових
   - **Count Plot** - хвороба серця за статтю
   - **Violin Plot** - розподіл віку у хворих і здорових

##  Про датасет

**Heart Disease Dataset** (UCI ID: 45) містить медичні дані 303 пацієнтів для прогнозування наявності серцевих захворювань.

### Характеристики датасету:
- **Multivariate**: Багатовимірний
- **Attribute Characteristics**: Categorical, Integer, Real
- **Number of Attributes**: 14 атрибутів (включно з цільовою змінною)
- **Has Missing Values**: Так (присутні пропуски в колонках ca та thal)
