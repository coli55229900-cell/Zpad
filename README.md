# Camera App

## Вимоги
- Ubuntu 20.04+
- Webcam
- cmake, g++, libopencv-dev

## Встановлення та запуск
./preinstall.sh
./build.sh
./run.sh

## Клавіші
| Клавіша | Дія |
|---------|-----|
| 1 | Normal |
| 2 | Invert |
| 3 | Blur |
| 4 | Canny |
| 5 | Sobel |
| 6 | Binary |
| 7 | Glitch |
| 8 | Picture-in-picture |
| Стрілки | Хрестик / поворот |
| +/- | Яскравість |
| ESC | Вихід |
| ЛКМ | Малювати лінії |
| Колесо | Зум |
